#### Idea

> This simple script can scan sunbnets and IP and provide a result in file.

#### Usage
> - create a script 
>
> - open terminal
```sh
cd /path to/IPscaner.sh
sudo chmod 775 IPscaner.sh
bash IPscaner.sh
```
